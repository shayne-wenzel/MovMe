Movime API Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description URL Endpoints Method Type Query Parameters Request Response
Return a list of All movies to the user /movies GET None None A JSON object holding data about all movies
Return data for a movie by title /movies/[title] GET :Title None JSON object holding data about the selected movie Example: 
                        
                          {
                            title: "Cloud Atlas",
                            year: "2012",
                            director: "The Wachowski sisters, Tom Tykwer",
                            genre: "Drama"
                          },
Return genre data by name/title (e.g., “Action”) /genres/[genre] GET :genre None A JSON object holding data about the selected genre. Example: 
                            
                                {
                                    name: 'Action',
                                    description: ''
                                }
Get a list of all users /users GET None None A JSON object holding data about all users, Example:
                            
                              name: "Margo",
                              favouriteMovies: ["Short Bus"],
                              id: 2
Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
                            
                                {
                                    "username": "",
                                    "email": "",
                                    "password": ""
                                    "D.O.B.": "",
                                    "favorites": []
                                }
Allow users to update their info /users/[username] PUT :Username A JSON object holding the new password. Example: 
                            
                              {
                                name: "James",
                                favouriteMovies: [X-Men],
                                id: 1
                              }
JSON object with the updated username. Example: 
                            
                              {
                                name: "James Dean",
                                favouriteMovies: [X-Men],
                                id: 1
                              }
Allow existing users to close account /users/[id] DELETE :id None A message indicating the user was deleted
Allow users to add a movie to their list of favorites /users/:user/favourites POST None A JSON object holding data about the movie to add. Example: 
                                
                                    {
                                        "username": "",
                                        "favorites": [Jackie Brown]
                                    }
A message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/:user/favourites DELETE favouritese None A message indicating the movie was deleted
